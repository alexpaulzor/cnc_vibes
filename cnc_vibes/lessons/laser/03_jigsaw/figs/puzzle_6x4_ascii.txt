6x4 jigsaw puzzle with square tabs

Convention:
- All internal vertical tabs are convex going RIGHT (A bumps into B, B into C, etc).
- All internal horizontal tabs are convex going DOWN (row 0 bumps into row 1, etc).
- Outer boundary is flat (no tabs on the perimeter of the whole puzzle).

Each piece is a 9-wide x 4-tall body. Tabs are 3-wide x 1-deep square notches
in the middle of each internal edge.

+----------+----------+----------+----------+----------+----------+
|          |          |          |          |          |          |
|   A0     |   B0     |   C0     |   D0     |   E0     |   F0     |
|          |          |          |          |          |          |
+-----+ +--+-----+ +--+-----+ +--+-----+ +--+-----+ +--+----------+
      | |        | |        | |        | |        | |
      +-+        +-+        +-+        +-+        +-+
+----------+----------+----------+----------+----------+----------+
|          |          |          |          |          |          |
|   A1     |   B1     |   C1     |   D1     |   E1     |   F1     |
|          |          |          |          |          |          |
+-----+ +--+-----+ +--+-----+ +--+-----+ +--+-----+ +--+----------+
      | |        | |        | |        | |        | |
      +-+        +-+        +-+        +-+        +-+
+----------+----------+----------+----------+----------+----------+
|          |          |          |          |          |          |
|   A2     |   B2     |   C2     |   D2     |   E2     |   F2     |
|          |          |          |          |          |          |
+-----+ +--+-----+ +--+-----+ +--+-----+ +--+-----+ +--+----------+
      | |        | |        | |        | |        | |
      +-+        +-+        +-+        +-+        +-+
+----------+----------+----------+----------+----------+----------+
|          |          |          |          |          |          |
|   A3     |   B3     |   C3     |   D3     |   E3     |   F3     |
|          |          |          |          |          |          |
+----------+----------+----------+----------+----------+----------+

Note: the vertical edges between rows (the | characters in the tab rows)
are intentionally omitted in this ASCII because they get visually
overwhelming. The tab notches (+--+ shapes) are the actual cut lines;
the piece interiors continue uninterrupted around them. The PNG version
(puzzle_6x4_with_letter_A.png) shows the polygons drawn correctly.

For the letter A overlay, see the PNG. Summary of overlay results:
  outside (11):   A0, A1, A2, A3, B0, E0, E1, F0, F1, F2, F3
  boundary (13):  B1, B2, B3, C0, C1, C2, C3, D0, D1, D2, D3, E2, E3
  inside (0):     none — the A's narrow strokes don't fully enclose any
                  cell at this granularity. Important finding for the
                  algorithm: every cell touching the letter must be
                  clipped or replaced; "use the inside cells as the
                  letter's sub-puzzle" doesn't apply.
